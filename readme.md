# Instalacion
- clonar el repo.
- instalar con

```
pnpm i
```

# Abrir server
Ejecutar 
```
pnpm dev:build
```

para construir el server y 

```
pnpm dev
```

para ejecutarlo

AMBOS COSAS DEBEN ESTAR CORRIENDO EN SIMULTANEO (abrite 2 consolas)

## Notas
Se abre por defecto en el puerto 4000
http://localhost:4000/

podes ir a la documentacion de la api:
http://localhost:4000/api-docs/
